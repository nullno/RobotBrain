## 舵机总览

本机器人共使用 **25 个** JOHO 串行总线舵机，通过 UART 总线串联连接，由 ESP32-S3 经 UART→CH340 驱动控制。

| 区域 | 舵机数量 | ID 范围 |
|------|----------|---------|
| 颈部 | 2 | 1 ~ 2 |
| 左臂 | 5 | 3 ~ 7 |
| 右臂 | 5 | 8 ~ 12 |
| 腰部 | 1 | 13 |
| 左腿 | 6 | 14 ~ 19 |
| 右腿 | 6 | 20 ~ 25 |
| **合计** | **25** | **1 ~ 25** |

## 装机步骤

### 1. 舵机 ID 设置

出厂舵机 ID 默认均为 1，**必须逐个修改 ID 后再串联**。

操作方法：
1. 单独连接一个舵机到 UART 总线
2. 在调试面板中使用「修改舵机 ID」功能
3. 或通过固件命令：`{"type": "set_servo_id", "old_id": 1, "new_id": <目标ID>}`
4. 修改完成后断电，连接下一个舵机

> ⚠️ **切勿将两个相同 ID 的舵机同时接入总线**，会导致通信冲突。

### 2. 机械安装顺序

建议安装顺序（从核心到外围）：

1. **腰部** → ID 13（大号舵机固定在底盘中心）
2. **左腿** → ID 14→15→16→17→18→19（从胯到脚）
3. **右腿** → ID 20→21→22→23→24→25
4. **颈部** → ID 1→2
5. **左臂** → ID 3→4→5→6→7（从肩到腕）
6. **右臂** → ID 8→9→10→11→12

### 3. 中位校准

每个舵机安装时：
1. 先通过软件将舵机设到中位值 2048（对应 ~180°）
2. 在中位状态下固定舵盘与机械臂
3. 使用「设置中位偏移」功能微调（固件命令 `set_middle_adjust`）

### 4. 串联接线

所有舵机通过 UART 总线串联：
- 第 1 个舵机接 ESP32 的 UART 输出
- 后续舵机从前一个舵机的通信口级联
- 建议每 8 个舵机为一段，段间加 100μF 去耦电容

## 默认角度与调试

### 站立姿态（默认中位值）

所有舵机初始位置为 **2048**（12 位范围 0-4095 的中间值）。

| 姿态 | 说明 |
|------|------|
| 直立站立 | 所有舵机 = 2048 |
| 手臂自然下垂 | ID 3,8 = 2048; ID 6,11 ≈ 1800（微屈） |
| 头部正前方 | ID 1,2 = 2048 |
| 双脚平行站立 | ID 14-19, 20-25 = 2048 |

### 舵机位置值与角度对应关系

| 位置值 | 角度 | 说明 |
|--------|------|------|
| 0 | 0° | 最小位置 |
| 1024 | ~90° | 四分之一 |
| 2048 | ~180° | 中位 |
| 3072 | ~270° | 四分之三 |
| 4095 | ~360° | 最大位置 |

> 实际有效旋转角度取决于舵机型号，通常为 0°~300° 有效行程。

### 调试建议

1. **单关节测试**: 使用调试面板「单舵机调试」Tab，逐个测试每个关节的运动方向和范围
2. **归零测试**: 点击「归零」按钮，确认所有关节回到直立姿态
3. **全身扫描**: 使用「扫描」功能确认 25 个舵机全部在线
4. **平衡测试**: 通过固件命令启用平衡算法 `{"type": "balance_enable", "enable": true}`

## 接线说明

### ESP32-S3 引脚分配

| 引脚 | 功能 | 说明 |
|------|------|------|
| GPIO 47 | UART TX | 舵机总线 TX（→ CH340 RX） |
| GPIO 48 | UART RX | 舵机总线 RX（← CH340 TX） |
| GPIO 20 | I2C SCL | IMU 时钟线 |
| GPIO 21 | I2C SDA | IMU 数据线 |

### 供电方案

| 模块 | 电压 | 说明 |
|------|------|------|
| ESP32-S3 | 3.3V / 5V USB | 逻辑供电 |
| 舵机总线 | 6V ~ 7.4V | **独立电源**，大电流（≥10A），建议 2S 锂电池 |
| IMU(MPU6050) | 3.3V | 从 ESP32 3.3V 取电 |

> ⚠️ **舵机必须使用独立电源**，禁止从 ESP32 USB 供电（电流不足会导致 ESP32 复位）。

### UART 串联示意

```
ESP32 UART TX/RX
        │
   ┌────┴────┐
   │ CH340   │  ← USB 转 TTL（已集成在控制板上）
   │ TX  RX  │
   └────┬────┘
        │
   ┌────┴────┐    ┌─────────┐    ┌─────────┐
   │ Servo 1 │───→│ Servo 2 │───→│ Servo 3 │───→ ...
   │ (ID=1)  │    │ (ID=2)  │    │ (ID=3)  │
   └─────────┘    └─────────┘    └─────────┘
```

## 注意事项

1. **电源**：舵机总线必须使用大电流独立电源（6~7.4V, ≥10A），全身 25 个舵机同时动作时峰值电流可达 8A+
2. **散热**：长时间运行平衡算法时，腿部舵机（ID 15,17,18,19,21,23,24,25）负载较大，注意监控温度
3. **机械限位**：安装前务必确认每个关节的物理限位，防止堵转烧舵机。可通过 `set_angle_limits` 命令设置软限位
4. **ID 冲突**：绝不能让两个相同 ID 的舵机同时在线，否则通信会紊乱
5. **固件更新**：修改固件后使用 `a_Firmware/flash.py` 脚本烧录，或参考 `a_Firmware/README.md`
6. **镜像安装**：左右对称关节（如左肩 ID3 和右肩 ID8）安装方向相反，注意运动方向的正负号
7. **IMU 安装**：MPU6050 必须水平安装在机器人胸腔中心，芯片面朝上，连线方向与机器人正前方一致
